# Phishing Email Analysis

## Objective

To identify phishing characteristics in a suspicious email sample.

## Tools Used

* Microsoft Word
* Sample Phishing Email

## Phishing Indicators Identified

1. Suspicious subject line
2. Urgent payment request
3. Suspicious link
4. PDF attachment
5. Generic greeting
6. Social engineering tactics
7. Possible email spoofing

## Conclusion

The email contains multiple phishing indicators and should be treated as suspicious. Users should avoid clicking links or downloading attachments without verification.

## Outcome

Learned how to identify phishing emails, recognize social engineering techniques, and analyze email threats.
